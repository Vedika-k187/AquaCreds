# AquaCreds 🌊

### Blockchain-Based Blue Carbon Registry & MRV System

AquaCreds is an **AI and blockchain-powered Monitoring, Reporting, and Verification (MRV) platform** designed to bring transparency and trust to **blue carbon ecosystems** such as mangroves, seagrass, and salt marshes.

The system integrates **AI models, satellite imagery, blockchain infrastructure, and decentralized storage** to verify ecosystem restoration projects and issue **tokenized carbon credits**.

🏆 **Smart India Hackathon 2025 – Finalist**

---

# Problem Statement

Blue carbon ecosystems store carbon significantly faster than terrestrial forests, but existing carbon credit verification systems face:

* Lack of transparency
* Slow manual verification
* Risk of greenwashing
* High verification costs

AquaCreds addresses these challenges using **ML-based verification and blockchain-backed transparency**.

---
## Key Features

- AI-based ecosystem monitoring using satellite imagery  
- Geotagged plantation data upload from mobile devices  
- Satellite validation to prevent fraudulent submissions  
- Blockchain-based carbon credit registry on Ethereum (Sepolia)  
- Tokenized carbon credit issuance using smart contracts  
- Decentralized storage of project data using IPFS
